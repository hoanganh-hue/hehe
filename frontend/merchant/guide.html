1 Bắt đầu với ZaloPay Merchant

Đăng ký tài khoản

  1. Truy cập trang đăng ký
  2. Chọn phương thức đăng ký (Google, Apple, hoặc Email)
  3. Điền thông tin doanh nghiệp
  4. Xác thực email và hoàn tất đăng ký
Lưu ý: Bạn cần chuẩn bị giấy phép kinh doanh và các giấy tờ pháp lý để xác thực tài khoản.

Xác thực tài khoản

Để sử dụng đầy đủ tính năng, bạn cần xác thực tài khoản:

  • Upload giấy phép kinh doanh
  • Cung cấp thông tin ngân hàng
  • Xác thực danh tính người đại diện
  • Chờ phê duyệt (1-3 ngày làm việc)

2 Thiết lập tài khoản

Thông tin doanh nghiệp

Điền đầy đủ thông tin doanh nghiệp để tăng độ tin cậy:

  • Tên công ty chính thức
  • Địa chỉ trụ sở
  • Số điện thoại liên hệ
  • Website (nếu có)
  • Mô tả hoạt động kinh doanh

Thiết lập ngân hàng

Kết nối tài khoản ngân hàng để nhận tiền:

  1. Vào Settings > Banking
  2. Thêm tài khoản ngân hàng
  3. Xác thực tài khoản (micro-deposit)
  4. Thiết lập lịch rút tiền

4 Tích hợp API

Lấy API Key

  1. Vào Settings > API Keys
  2. Click "Create New Key"
  3. Đặt tên cho API key
  4. Chọn quyền truy cập (Read/Write)
  5. Lưu API key an toàn
Cảnh báo: Không chia sẻ API key với bên thứ ba. Nếu bị lộ, hãy tạo key mới ngay lập tức.

Tích hợp cơ bản

Ví dụ tích hợp với JavaScript:

// Khởi tạo ZaloPay Merchant
const zalopay = new ZaloPayMerchant({
    apiKey: 'your-api-key',
    environment: 'sandbox' // hoặc 'production'
});

// Tạo giao dịch
const transaction = await zalopay.createTransaction({
    amount: 100000,
    currency: 'VND',
    description: 'Thanh toán đơn hàng #123',
    returnUrl: 'https://yoursite.com/success',
    cancelUrl: 'https://yoursite.com/cancel'
});

Webhook

Thiết lập webhook để nhận thông báo real-time:

  1. Vào Settings > Webhooks
  2. Thêm URL endpoint
  3. Chọn events cần lắng nghe
  4. Test webhook

5 Sử dụng Dashboard

Tổng quan Dashboard

Dashboard cung cấp cái nhìn tổng quan về hoạt động kinh doanh:

  • Doanh thu: Tổng doanh thu theo ngày/tuần/tháng
  • Giao dịch: Số lượng và tỷ lệ thành công
  • Phương thức: Phân bổ theo phương thức thanh toán
  • Xu hướng: Biểu đồ xu hướng giao dịch

Quản lý giao dịch

Trong mục Transactions, bạn có thể:

  • Xem danh sách tất cả giao dịch
  • Lọc theo trạng thái, thời gian, phương thức
  • Xuất báo cáo Excel/CSV
  • Xem chi tiết từng giao dịch
  • Hoàn tiền (refund) nếu cần

6 Báo cáo & Thống kê

Báo cáo doanh thu

Tạo báo cáo chi tiết về doanh thu:

  1. Vào Reports > Revenue
  2. Chọn khoảng thời gian
  3. Chọn phương thức thanh toán
  4. Xuất báo cáo PDF/Excel

Phân tích xu hướng

Sử dụng các biểu đồ để phân tích:

  • Xu hướng giao dịch theo thời gian
  • Phân bổ theo phương thức thanh toán
  • So sánh doanh thu các tháng
  • Tỷ lệ thành công/thất bại

7 Bảo mật

Bảo mật tài khoản

  • Sử dụng mật khẩu mạnh
  • Bật xác thực 2 lớp (2FA)
  • Thường xuyên thay đổi API key
  • Không chia sẻ thông tin đăng nhập
  • Kiểm tra log đăng nhập thường xuyên

Bảo mật giao dịch

  • Sử dụng HTTPS cho website
  • Mã hóa dữ liệu nhạy cảm
  • Validate dữ liệu đầu vào
  • Sử dụng webhook để xác thực
  • Giám sát giao dịch bất thường

8 Xử lý sự cố

Giao dịch thất bại

Nguyên nhân thường gặp:

  • Thông tin thẻ không chính xác
  • Thẻ bị khóa hoặc hết hạn
  • Số dư không đủ
  • Lỗi kết nối mạng

Giải pháp:

  • Kiểm tra thông tin thẻ
  • Liên hệ ngân hàng phát hành thẻ
  • Thử phương thức thanh toán khác
  • Kiểm tra kết nối mạng

Lỗi tích hợp API

Kiểm tra:

  • API key có đúng không
  • URL endpoint có chính xác không
  • Format dữ liệu có đúng không
  • Headers có đầy đủ không
